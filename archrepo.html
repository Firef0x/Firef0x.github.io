Guide to Add This Repo
   添加本仓库指引

Add the repo to /etc/pacman.conf under the current official Arch Linux repos:

在 /etc/pacman.conf 文件中现有的 Arch Linux 官方仓库下添加仓库信息：

[firef0x]
Server = http://firef0x.github.io/archrepo/$arch
SigLevel = Never

[home_firef0x_Arch_Extra]
Server = http://download.opensuse.org/repositories/home:/firef0x/Arch_Extra/$arch
# Mirror sites for Chinese users 中文用户镜像仓库 中文用戶鏡像倉庫
# Server = ftp://ftp.twaren.net/Linux/OpenSuSE/repositories/home:/firef0x/Arch_Extra/$arch
# Or 或
# Server = http://ftp.yzu.edu.tw/Linux/openSUSE/repositories/home:/firef0x/Arch_Extra/$arch
SigLevel = Never

Then, update your sync database and install the packages:

然后，更新您的远程同步数据库并安装包：

$ sudo pacman -Syu [packages ...]

Troubleshooting
   疑难解答

Failed to install packages, report "Maximum file size exceeded" or "Invalid or corrupted package (checksum)"
安装软件包失败，报错 "Maximum file size exceeded" 或 "Invalid or corrupted package (checksum)"

Well, the repository site of openSUSE looks like not friendly to Arch Linux users. Please setup wget as default download tool by removing comment from the following line in /etc/pacman.conf:

好吧，openSUSE 的软件仓库似乎对 Arch Linux 用户不太友好。请通过取消 /etc/pacman.conf 中的这行注释以将 wget 设置为默认下载工具： 

XferCommand = /usr/bin/wget --passive-ftp -c -O%o %u

Then try again.

然后再试一次。

If not work, please try the mirror sites for Chinese users listed above.

如果仍不成功，请尝试使用上面列出的中文用户镜像仓库。
